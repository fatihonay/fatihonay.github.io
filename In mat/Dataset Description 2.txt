The pre-processed EEG data matrix can be loaded by MATLAB. The read-out struct contains 43 field. 'Setname' is number of participant, 'nbchan' is number of EEG channel, 'trails' is number of trails, 'pnts' is number of time points, 'times' is the actural time, 'data' is the EEG data of the whole expirement of 34 channels. 'Event' contians three cell, 'type' is the type of stimulus, 'latency' is the time when one stimuli happen, 'urevent' is the number of stimulus. 'EVENTLIST' contains the number of each stimuli and actual time when stimuli happened.
